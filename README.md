# 番茄钟应用

一个基于Web的番茄工作法计时器应用，帮助你提高工作效率和时间管理能力。

## 功能特点

### 基础功能
- 倒计时器：默认25分钟工作时间，5分钟休息时间，可自定义时长
- 开始/暂停/重置按钮：控制计时器状态
- 工作/休息模式切换：自动或手动切换工作与休息状态

### 亮点功能
- 任务列表：可添加、删除、标记完成任务，并统计完成情况
- 背景音效：专注时可播放白噪音（雨声、咖啡厅、森林、海浪），提高专注力
- 自动开始下一个周期：可选择在当前周期结束后自动开始下一个周期
- 今日统计：记录并显示今日完成的番茄数量
- 数据持久化：任务列表和今日统计会保存在本地存储中

## 技术实现

- HTML5 + CSS3 + JavaScript
- Tailwind CSS v3 用于样式设计
- Font Awesome 用于图标
- Web Audio API 用于音频播放
- LocalStorage API 用于数据持久化

## 如何使用

1. 打开 `index.html` 文件即可开始使用
2. 调整工作时间和休息时间（可选）
3. 添加任务到任务列表（可选）
4. 点击"开始"按钮启动计时器
5. 专注工作直到计时器响起，然后休息
6. 重复上述过程

## 自定义设置

- **工作/休息时间**：通过左侧面板的加减按钮或直接输入数字来自定义时长
- **自动开始**：勾选"自动开始下一个周期"选项，让计时器在当前周期结束后自动开始下一个周期
- **背景音效**：选择喜欢的背景音效，并调整音量大小

## 任务管理

- 点击"+"按钮添加新任务
- 点击任务前的方框标记任务为已完成/未完成
- 点击任务文本将其设为当前活动任务
- 点击垃圾桶图标删除任务

## 浏览器兼容性

- Chrome

- Safari
- Edge
